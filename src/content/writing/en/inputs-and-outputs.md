How do inputs turn into outputs?

It always starts with **input devices**, which take input from the outside world and convert it into binary data. That data is stored in memory. Then the CPU performs all the computations and instructions on it. The binary results are then converted into physical output by **output devices**.

## A practical example

When you press the letter "C" on your keyboard, the keyboard converts the letter to a number, and that number is sent as binary data into the computer.

The CPU then calculates how to display the letter "C" in pixels. It requests instructions from memory on how to draw this character, performs them step by step, and stores the result back into memory. That result is sent as binary data to the screen — an output device — which converts it into illuminated dots that make up what you see.

This all happens instantaneously, but inside the computer thousands of instructions are being executed.

The more complicated the inputs and outputs, the more data is involved, and the more processing power and memory the computer needs.
