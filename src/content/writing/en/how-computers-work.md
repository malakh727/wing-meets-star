Wherever you look, you'll find computers: on your desk, in your pocket, everywhere. Computers have essentially helped and continue to help improve human lives, making them more efficient and faster.

Although we use them every day, we rarely stop to ask: what is a computer, and how does it work?

A computer has four basic operations: taking input, storing information, processing it, and outputting the result.

Computers started out as basic calculators that only processed numbers. Now we use them for far more complex things: playing games, controlling robots, communicating with each other.

Modern computers look nothing like the old ones, but they still perform the same four basic operations.

A computer is an electronic device that requires electricity to work. It consists of smaller components, each responsible for one of the four basic operations. Understanding those components is how we understand the computer.
