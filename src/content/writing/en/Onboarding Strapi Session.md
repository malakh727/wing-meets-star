# Onboarding Strapi Session

# What is Strapi?

- A user-friendly CMS (Content Management System)
- Allows you to manage and publish website/app content without needing a developer
- Works through a clean and intuitive interface

# **Logging In**

- Access the Strapi admin panel via a secure link
- Use your assigned credentials
- Reset password available if needed

# **Dashboard Overview**

- Main navigation: Collection Types, Media Library
- What each section is used for
- How to find what you need quickly

# Media Library

- Upload images, videos
- Search and organize media files
- Best practices for naming and uploading assets

# **Collection Types**

- Think of them as content categories (e.g., Blog Posts, Pages, Services)
- Each type has its own fields (title, image, description, etc.)

**Main actions:**

- Create new entries
- Edit existing content
- Save and publish or save as draft

# **Helpful Tips**

- Use consistent formatting (titles, descriptions)
- Preview before publishing (if preview is available)
- Keep SEO in mind when entering titles and descriptions and alternative texts
