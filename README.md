# 🌌 Wing Meet Star

A personal knowledge garden, blog, and portfolio.

---

## ✨ Vision

A space where:

- ideas grow over time
- knowledge connects like constellations
- engineering meets curiosity
- writing meets exploration

---

## 🧭 Features

### 📝 Blog (Essays)

Long-form writing.

---

### 🌱 Notes

Short evolving ideas.

---

### ⭐ Star Stories

Creative storytelling with future interactive star map.

---

### 💻 Portfolio

Projects, skills, experience.

---

## 🌍 Languages

- English → default (no URL prefix)
- Arabic → `/ar/` (RTL)
- Japanese → future

Example:

- `/blog`
- `/ar/blog`

---

## 🧱 Tech Stack

- Astro
- React
- TailwindCSS
- Markdown / MDX
- GitHub Pages
